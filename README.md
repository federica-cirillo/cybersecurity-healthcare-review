# Cybersecurity in Healthcare: Challenges, Impacts, and Solutions

**University Exam:** Health Information Technologies
**Type:** Team work

## Overview

This project is a literature-based presentation examining cybersecurity risks in the healthcare sector, covering cyber threats, their impact on care quality and patient trust, and current/emerging strategies to prevent data breaches.

## Key Topics

- **CIA Triad** (Confidentiality, Integrity, Availability) as the foundation of information security
- **Main cyber threats**: ransomware (e.g. WannaCry, 2017 UK NHS attack), phishing (>90% of breaches), unauthorized access, DDoS
- **Vulnerable medical devices**: insulin pumps (Medtronic/FDA case) and pacemakers, including attack models (DoS, Man-in-the-Middle, down-sampling, control-parameter attacks, coordinated attacks)
- **Impact on healthcare**: direct consequences for patient safety (e.g. AOU Verona, Oct 2023), loss of patient trust, economic/legal costs
- **Defense technologies**: firewalls, Intrusion Detection Systems (NIDS/HIDS), AI/Machine Learning for threat detection and prediction, cloud security
- **Cloud computing in healthcare**: hybrid cloud model, performance, reliability, security, cost reduction (e.g. AWS HealthOmics, HealthImaging, HealthScribe)
